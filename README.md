# 局域网文件共享工具

一个跨平台的局域网文件共享工具，在同一局域网内的任何设备（电脑、手机、平板）用浏览器即可浏览和下载文件。

## 特性

- **跨平台** — Windows / macOS / Linux 都能运行
- **GUI 界面** — 图形化操作，添加/移除共享文件夹，启停管理
- **多文件夹** — 同时共享多个不同路径的文件夹，各自独立端口
- **零客户端** — 对方只需浏览器打开地址，不需要安装任何软件
- **自动保存** — 关闭后记住共享列表，下次打开自动恢复

## 系统要求

- Python 3.7+
- tkinter（Python 自带，无需额外安装）

## 快速开始

### Windows

双击 `启动共享.bat`

或者命令行：

```
python lan_share.py
```

### macOS / Linux

```
python3 lan_share.py
```

或双击 `启动共享.sh`

## 使用方法

1. 启动后出现 GUI 窗口
2. 点击 **＋ 添加文件夹**，选择要共享的目录（支持不同盘符）
3. 点击 **▶ 全部启动**，所有文件夹开始共享
4. 把访问地址（如 `http://192.168.1.100:8000`）发给局域网内的其他人
5. 对方在浏览器打开即可浏览和下载文件

### 其他操作

| 操作 | 说明 |
|------|------|
| 双击某行 | 复制访问地址到剪贴板 |
| 启动选中 / 停止选中 | 单独控制某个共享 |
| 修改端口 | 先停止该共享，再修改端口号 |
| 打开文件夹 | 在资源管理器中打开该共享目录 |
| 删除选中 | 从列表中移除该共享 |

### 文件页面功能

- 点击文件夹进入子目录
- 点击文件名直接下载
- 顶部搜索框可过滤文件名
- 支持面包屑导航

## 文件说明

```
├── lan_share.py        # 主程序
├── 启动共享.bat         # Windows 启动脚本
├── 启动共享.sh          # macOS/Linux 启动脚本
└── README.md           # 本文件
```

启动后会自动生成 `lan_share_config.json` 保存共享列表配置。

## 注意事项

- 确保双方在同一局域网下（连接同一个路由器/Wi-Fi）
- 如果对方无法访问，检查本机防火墙是否拦截了对应端口
- 端口范围默认从 8000 开始递增，可手动修改
- 文件较大时下载速度取决于局域网带宽，与互联网速度无关
